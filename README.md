# Pulse

A unified iOS tracker for money, movement, and daily rituals — with an AI companion.

## Stack

- SwiftUI (iOS 17+)
- SwiftData + CloudKit
- HealthKit, WidgetKit, ActivityKit
- Anthropic API (Claude Haiku) via proxy

See [STACK.md](STACK.md) for full architecture and dev setup decisions.
